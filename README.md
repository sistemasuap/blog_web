## ♥ Sobre el proyecto

Página web interactiva que narra la historia de amor de **Elmer Marcelino De la Paz Ttito** y **María Isabel Campos Taipe** — desde su encuentro, su boda en 2021, sus tres hijos, su hogar y sus viajes por el mundo. Construida con HTML, CSS y JavaScript puros, sin frameworks, con foco en animaciones elegantes y una experiencia visual memorable.

---

## ✨ Características

- **Header animado** con tipografía Cormorant Garamond y efecto de pulso dorado en el ampersand
- **Cursor personalizado** con anillo de seguimiento suave
- **Barra de progreso** de scroll en la parte superior
- **Estrellas generadas** dinámicamente con efecto de parpadeo
- **Pétalos cayendo** animados en el fondo
- **Timeline interactiva** con 5 momentos clave de su historia
- **Estadísticas animadas** — los números cuentan solos al entrar en pantalla
- **Contador en vivo** que muestra exactamente cuántos días, horas, minutos y segundos llevan casados
- **Cards de recuerdos** con hover effects y gradientes oscuros
- **Scroll reveal** — cada sección aparece con animación al hacer scroll
- **Diseño responsive** adaptado a móviles

## 🚀 Cómo usarlo

**1. Clona o descarga el repositorio**
```bash
git clone https://github.com/gecamepsis/elmer-maria-isabel.git
```

**2. Entra a la carpeta**
```bash
cd elmer-maria-isabel
```

**3. Abre el archivo en tu navegador**
```bash
# Opción A — doble clic en index.html
# Opción B — con Live Server en VS Code
# Opción C — desde terminal
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

> ⚠️ Los 3 archivos deben estar en la **misma carpeta** para que los links funcionen correctamente.
